# 🖼️ Image Annotation Tool

A simple and interactive **Image Annotation Tool** built with **Python** and **OpenCV** that allows users to annotate images by drawing bounding boxes. This project is useful for creating datasets for computer vision and object detection tasks.

---

## ✨ Features

* 📂 Load images from your local system
* 🖱️ Draw bounding boxes using the mouse
* 📝 Save annotation coordinates
* 🎯 Simple and beginner-friendly interface
* ⚡ Fast annotation workflow
* 🐍 Built entirely with Python & OpenCV

---

## 🛠️ Tech Stack

* Python 3.x
* OpenCV
* NumPy
* Jupyter Notebook

## 🚀 Installation

### 1. Clone the repository

```bash
git clone https://github.com/palakgupta29/Image-Annotation.git
cd Image-Annotation
```

### 2. Create a virtual environment (Optional)

```bash
python -m venv venv
```

Activate it:

**Windows**

```bash
venv\Scripts\activate
```

**Linux / macOS**

```bash
source venv/bin/activate
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

Or install manually:

```bash
pip install opencv-python numpy
```

---

## ▶️ Usage

Run the notebook:

```bash
jupyter notebook annotation.ipynb
```

Or open it directly in **VS Code** and execute the cells.

## 📸 Sample Workflow

1. Open an image.
2. Draw a bounding box around the object.
3. Save the annotation.
4. Repeat for remaining images.
5. Export the annotations for model training.
